==============
LangmuirPython
==============

LangmuirPython is a set of python modules and python scripts that aid in the use of
the Langmuir charge transport simulation code.

===============
Getting Started
===============

LangmuirPython depends on the following python modules, which you should install:

* numpy
* scipy
* pandas
* quantities
* matplotlib
